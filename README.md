### 使用方法
1.  **打开终端**：您那高贵华丽时尚帅气简约透明云母亚克力仙女下凡阳光彩虹重金属蒸汽波赛博朋克极简主义超未来...（编不下去了）的终端。
2. 手指在键盘上如蝴蝶般翩然起舞，敲下`cd /usr/local/bin/`，这一刻，整个太阳系的智慧生命都为之惊叹
3. 接着输入`git clone https://github.com/Oiiai/apt-in-archlinux.git`，回车
   如果您嫌等待时间过长，可以在短暂的100琥珀纪内思考该项目的意义，我相信您一定做得到
4. 进入工作目录下的`apt-in-archlinux/`文件夹（这是不小心执行第3步的后果）
5. 找到`apt`文件，并把它剪贴（或者复制）到上级目录（也就是`/usr/local/bin/`）`cp apt /usr/local/bin/`
6. *诶！这时候屏幕前的您可能会发出疑问：为什么不用`mv`而使用`cp`指令呢 因为这样就能消耗您键盘上c、p两个案件的寿命，让它们早日正寝*
